卑贱的男人

发表于 2017-11-09 | 分类于 观后感 | | 阅读

看完“驴得水”感触很深，这部电影的深意超越了大多数商业电影，既描绘了人性善良温和的一面，又揭露了贪婪虚伪狠毒的一面。天使与魔鬼之间只差一个价码，唯一保持单纯的是两个女孩子。

影片中每个人都有自己的个性特征，在前半部分，所有的男人都是善良的正直的有梦想的，愿意为农村教育事业奉献的。当金钱利益摆在眼前的时候，却失去了这宝贵的信念，卑贱得降低自己一层层的人格底线。 校长大力为了得到更多的资金，向教育局谎报了并不存在的驴得水老师，为了维护这个谎言，却又不得不用更多更荒谬的谎言去弥补，甚至出现了让铜匠假扮老师，用本地方言假冒英语，让铜匠假装尸体而突然死而复生的荒诞情景。

张一曼是电影中唯一的女性教师，也是最重要的角色，每位男性在开始时都是保护张一曼的，裴魁山对张一曼表达了爱意，校长大力和张一曼在浪漫的夜色中跳舞，铜匠在离开的时候，给张一曼唱情歌，周铁男在特派员面前奋力维护张一曼。而也是同样这群男人把张一曼拖向了深渊。除了明确的主线外，电影用微妙的镜头描绘了这些卑微的人。裴魁山在没有得到张一曼之后，对她破口大骂，并显露出其贪婪的嘴脸；校长为了保护他的学校，用剪刀剪去了一曼的头发；铁男为了保住自己的性命，在保安队长准备强奸一曼的时候懦弱得蜷缩在墙角；铜匠为了发泄自己的怒气，让其他老师辱骂一曼，让校长剪掉她的头发。

可怕的是这个故事让我觉得很真实，国人特别是男性，很多没有自己的底线，为了金钱利益，可以抛弃自己的底线，让生活充满了虚假的谎言。当美国人伸手到包里拿出一万美金时，旁边的中国人眼睛里都闪烁着贪婪的光。崇洋媚外，当美国人问校长有什么需求时，校长把女儿推出去，说自己的女儿想去美国，假扮尸体的铜匠也突然起身要求去美国。

看完电影之后，内心不能平静，因为这个故事涉及了中国社会最重要的方面，政府和教育还有社会价值观。腐败的政府，层层克扣贫困乡村的教育基金；畸形的教育观，孩子的教育只是为了获得钱财的一个借口；畸形的社会价值观，以金钱来衡量自己的价值。现在的中国不缺钱，却是非常稀缺诚实正直的价值观。

最后想以优美又带点忧郁的“我要你”歌词作为结束，这首歌有李健风格，李健也确实在演唱会上唱过这首歌，后来才知道是根据《夜来香》改编老狼演唱的歌曲，这首歌让我想拿起吉他弹奏这美妙的旋律。
